# Aviator Circle Demo

A modern Aviator-style circle demo game built using HTML, CSS, and JavaScript.

## Features

- Virtual coins balance
- Circular plane animation
- Bet amount input
- Auto cashout
- Manual cashout
- Round history
- Responsive interface
- Developed by Iradi

## Disclaimer

This project is for educational and demonstration purposes only. It uses virtual coins and does not involve real money, betting, or gambling.
